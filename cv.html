Tóm tắt

Lập trình viên định hướng backend và lập trình mạng. Ưu tiên code rõ ràng, an toàn, hiệu năng; quen làm việc với Java và JavaScript/Node.js, có kiến thức nền tảng về giao thức mạng.

Kinh nghiệm/ Dự án

Mini Server Java xử lý socket

  • Thiết kế server TCP dùng thread pool (ExecutorService) phục vụ nhiều client.
  • Áp dụng synchronized/Lock đảm bảo an toàn dữ liệu chia sẻ.

REST API quản lý thiết bị mạng (Node.js)

  • Xây dựng API CRUD, xác thực cơ bản, phân trang.
  • Viết script npm để tự động hoá tác vụ phát triển.

Demo JDBC

  • Kết nối DB qua JDBC, sử dụng PreparedStatement và transaction.
  • Viết truy vấn tối giản và ánh xạ kết quả.
